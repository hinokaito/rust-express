# 補足

## ユニット構造体
フィールドを持たない構造体を名前空間としてよく使います。

```rust
struct Console;

impl Console {
    // ...
}

fn main() {
    // Console::○○();
    // Console::△△();
}
```

## タプル構造体
プリミティブ型(標準で用意されている型)にドメインを与えることで可読性や型安全性を高めます。

```rust
struct Meters(f64);
struct Seconds(f64);

fn speed(distance: Meters, time: Seconds) -> u64 {
    distance.0 / time.0 // タプルなので.0でアクセス
}
```
