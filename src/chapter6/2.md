# 構造体
構造体は、複数の値をひとまとめにして表現するためのユーザー定義型です。各値をフィールドと呼びます。

```rust
struct User {
    name: String, // String型のnameというフィールド
    age: u32,
    active: bool,
}

fn main() {
    // インスタンス生成
    let taro = User {
        name: String::from("taro"),
        age: 20,
        active: true, 
    };

    if taro.active {
        println!("{}はアクティブです。", taro.name);
    }
}
```

enumやstructをフィールドにする
```rust
#[derive(Debug)] // 表示させるために使う
enum ShopType {
    Convenience,
    Pharmacy,
    Book,
    Flower,
}

#[derive(Debug)] // 表示させるために使う
struct Shop {
    shop_type: ShopType,
    name: String,
}

fn main() {
    // フィールド単位ではなく、インスタンス単位でしか可変/不変を指定できない
    let mut shop1 = Shop {
        shop_type: ShopType::Convenience,
        name: String::from("セブンイレブン"),
    };

    shop1.name = String::from("ファミリーマート");

    println!("{:#?}", shop1); // :#? で整形して表示してくれる
}
```
