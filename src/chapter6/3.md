# メソッド実装
構造体や列挙型には`impl`キーワードでメソッドを実装できます。

```rust
struct User {
    name: String,
    age: u32,
}

impl User {
    fn introduction(&self) {
        println!("こんにちは。{}歳の{}です。", self.age, self.name);
    }
}

fn main() {
    let kaito = User {
        name: String::from("快音"),
        age: 19,
    };

    kaito.introduction();
}
```

同じ要領で、コンストラクタも作れます。

```rust
#[derive(Debug)]
struct User {
    name: String,
    age: u32,
}

impl User {
    fn new(name: String, age: u32) -> Self { // Selfは自身の型。この場合User
        Self { // 変数名とフィールド名が一致する場合、変数だけでOK。
            name,
            age
        }
    }
}

fn main() {
    // インスタンスから呼ば差ない場合、構造体::メソッド()
    let taro = User::new(String::from("太郎"), 18);

    println!("{:#?}", taro);
}
```
