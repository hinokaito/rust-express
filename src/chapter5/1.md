# 固定長配列
固定長配列は、柔軟性こそありませんが、スタックに積まれるので非常に高速です。単一の型のみ入れることができます。大きな配列には向いていません。コンパイル時にサイズが確定します。

型は`[要素の型; 要素数]`

```rust
fn main() {
    // let 変数名: 型 でその変数の型を明示できる。
    // 推論されるケースもあれば、明示必須なケースも多い。
    let array: [i32; 5] = [0, 1, 2, 3, 4];

    println!("{}", array[2]); // 2
}
```

## メソッド
```rust
fn main() {
    let array: [i32; 5] = [1, 2, 3, 4, 5];

    // 長さ
    println!("{}", array.len());
    // 最初の要素
    println!("{:?}", array.first());
    // 最後の要素
    println!("{:?}", array.last());
}
```
