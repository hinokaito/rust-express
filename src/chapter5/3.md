# スライス
厳密にはコレクションではありませんが、コレクションの一部分を参照したデータ型です。所有権を持ちません。

文字列`&str`も内部的にスライスです。

```rust
fn main() {
    let s = String::from("my name is kaito");

    let name = &s[11..]; // index11から最後まで
    println!("{}", name); // kaito

    let initial = name.chars().next().unwrap();
    println!("{}", initial) // k
}
```
