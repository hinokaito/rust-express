# ベクタ
多分一番使います。

可変長で、実行時にサイズが決まります。要素はヒープに格納されます。

保持される要素は3つです。
1. ヒープへのポインタ
2. 要素数
3. 保持できるサイズ

```rust
fn main() {
    // vec!マクロで初期値ありで生成できる
    let mut v: Vec<i32> = vec![0, 1, 2, 3, 4];
    println!("{:?}", v); // [0, 1, 2, 3, 4]

    v.push(5); // 要素を後ろに追加
    println!("{:?}", v); // [0, 1, 2, 3, 4, 5]

    v.remove(5); // 一番後ろの要素を取り出す
    println!("{:?}", v); // [0, 1, 2, 3, 4]

    println!("{}", v.len()); // 5
}
```
