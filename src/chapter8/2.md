# トレイト
実はこれまでで使いました。
`#[derive(Debug)]`
これは、対象の構造体等にDebugというトレイトを実装させています。


## 基本的な使用例
```rust
// トレイト定義
trait Greet {
    // 実装先はgreetメソッドを持つ必要があると宣言
    fn greet(&self);
}

struct Person {
    name: String,
}

// impl トレイト for で実装できる
impl Greet for Person {
    // トレイト定義時のメソッドをすべて取り入れる
    fn greet(&self) {
        println!("こんにちは、{}です", self.name);
    }
}

fn main() {
    let taro = Person {
        name: "太郎".to_string(),
    };

    taro.greet();
}
```

## トレイトを引数に使う
```rust
trait Greet {
    fn greet(&self);
}

struct Person {
    name: String,
}

impl Greet for Person {
    fn greet(&self) {
        println!("こんにちは、{}です", self.name);
    }
}

// Greetを実装している型なら何でもOK
fn say_hello(item: impl Greet) {
    item.greet();
}

// ジェネリクス版
fn say_hello2<T: Greet>(item: T) {
    item.greet();
}

fn main() {
    let taro = Person {
        name: "太郎".to_string(),
    };
    let jun = Person {
        name: "ジュン".to_string(),
    };
    say_hello(taro);
    say_hello2(jun);
}
```

## デフォルト実装
実装を持たせられます。
```rust
trait Greet {
    fn greet(&self) {
        println!("こんにちは");
    }
}

struct Person;

impl Greet for Person {}

fn main() {
    let taro = Person;
    taro.greet()
}
```
