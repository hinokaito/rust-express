# if
`if`キーワードは、条件分岐に使います。

```rust
fn main() {
    let x = false;

    if x {
        println!("trueでした")
    } else {
        println!("falseでした")
    }
}
```

`if`は他言語と異なり、 **式** であることがポイントで、値を返せます。

```rust
fn main() {
    let answer = true;

    println!("{}", if answer { "合っています" } else { "間違っています" });
}
```
