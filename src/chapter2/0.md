## 補足
今書くべきかわかりませんが、Rustにおける`;`の有無は、文か式かの違いがあります。

式は、値を返してスコープを抜けます。

```rust
// これは式なので、1+2が処理された瞬間に値を返しながらスコープを抜けます。
{
    1 + 2
}

// これは文なので、1+2の処理が終わった後、何もないので何も返さずにスコープを抜けます。
{
    1 + 2;
}
```

Rustの特色です。理解していないとコードが増えたり意図しない動きをします。
