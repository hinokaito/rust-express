# print
rustで文字を出力するには`println`/`print`マクロを使います。

> [!NOTE]
> 最後に!のついたものはマクロです。マクロは、「コードを生成するためのコード」で、作るには上級者向けな機能ですが、使う分には非常に楽で便利な機能です。println!など、標準でいくつか提供されています。

先ほど作成したプロジェクトの中に`src`フォルダがあると思います。その中に`main.rs`があるので開いてください。

そのコードをすべて消して以下のコードを入れてください。

次回からファイルの説明はしないので好きにやってください。ちなみに、本書のコード部分にカーソルを合わせて右上の実行ボタンを押せば実行できます。

```rust
// main関数がエントリーポイント
// 必ず関数の上に処理を書く
fn main() {
    // 改行無しの出力
    print!("Hello ");

    // 改行あり(内部的に最後に\nが付く)の出力
    println!("Rust"); // セミコロン忘れずに
    
    // 値を埋め込む
    println!("数字は{}です。", 5); // 値は勝手に文字になる

    // 文字以外の型を埋め込む
    println!("{:?}です。", true); // :? が必要
}
```
