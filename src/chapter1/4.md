# 環境構築
## Rust
少々複雑なので、参考サイトURLを張ります。

[参考サイト](https://utakamo.com/article/rust/install/index.html)
Windowsのセクションを見てください。

Rustのインストーラーは、DOWNLOAD RUSTUP-INIT.EXE(X64)を選んでください。

分からなければ聞いてください。GeminiやGPTでも多分いける。

```bash
rustc --version
```

これでバージョンが表示されればRustのインストールは成功です。

## VSC
VSCで拡張機能の検索に、`rust-analyzer`と入力して出てきたものをインストールしてください。
