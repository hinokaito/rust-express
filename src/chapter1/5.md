# Cargo - Hello, World
プロジェクトを作成して、"Hello, World!"を出力しましょう。PowerShellで以下の通り実行してください。
なお、プロジェクト名はスネークケース[^1]である必要があります。
```shell
cd C:/; mkdir dev/rust #C:/dev/rustフォルダを作成
cd dev/rust

# cargo new コマンドでプロジェクト作成
cargo new first_project
cd first_project
```

```shell
cargo run
> Hello, World!
```

## cargoコマンド
cargoはRustのビルドシステム兼パッケージマネージャです。最低限、以下のコマンドは抑える必要があります。やっていればなれますが、今の段階で上位3つは必ず押さえてください。

|コマンド|意味|
|---|---|
|cargon new|新規プロジェクト|
|cargo build|ビルド|
|cargo run|ビルド&実行|
|cargo build –release|最適化されたビルド|
|cargo run –release|最適化されたビルド&実行|
|cargo check|コンパイルに通して誤りがないかチェック|
|cargo init|カレントディレクトリをrustプロジェクト化|
|cargo add|依存関係を追加|
|cargo update|依存関係を更新|
|cargo fmt|コードを整えてくれる|
|cargo clippy|コードの改善提案をしてくれる|

[^1]: スネークケース: `snake_case`,`rust_app`など、単語をアンダースコアで区切る命名規則です。
