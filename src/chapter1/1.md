# なぜRust
結論: 速い・安全・並行処理

TIOBE Indexという、プログラミング言語を「どれだけ検索・言及されているか」でランキングにしたサイトがあります。当然、歴史のある言語が有利です。<br>
Rustを見ると、100位中13位です。これは異常です。なぜならRustは2015年に正式リリースされたからです。

|Ranking|Language|Release year|
|---|---|---|
|1|Python|1991|
|2|C|1989|
|3|Java|1996|
|4|C++|1998|
|5|C#|2002|

本書でRustの本質的な強みは深く説明しません。テーマは「速習」だからです。<br>
ただし、抽象的に「強さ」は一瞬で証明できます。

世界のOSはWindows, MacOS, Linuxが9割9分9厘を占めていますが、

- Linuxは先月Rustを正式に採用することを決定しました。
- Windows開発者は「2030年までにCとC++のコードベース全体をRustに置き換えたい」とコメントしました。

すでにOSではC,C++の立場を奪っています。

RustはC,C++と違って、非常に広い分野で採用されます。チャットアプリも例に洩れません。

本書では、3日でRustを習得することを目指します。
