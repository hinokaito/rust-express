# Result

`Result`型は、「成功したかエラーだったか」「エラーだった場合なぜエラーが起きたのか」を表現できます。

成功した場合`Ok()`、失敗した場合`Err()`です。

`Option`と同じように剥がす必要があります。

`Option`もそうですが、使用するメソッドの多くは`Result`型か`Option`型で返ってきます。

```rust
fn convert_and_show(t: &str) { 
    // 文字列から数値に変更したい
    match t.parse::<i32>() {
        Ok(n) => println!("成功! : {}", n),
        Err(e) => println!("失敗 : {}", e)
    }
}

fn main() {
    let x = "32";
    let y = "abc";

    convert_and_show(x); // 成功! : 32
    convert_and_show(y); // 失敗 : invalid digit found in string
}
```
