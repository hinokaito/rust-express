# 所有権
値には、必ず所有者がいます。所有者がその値を持てる権利を「所有権」と言います。
所有権は他者に渡すことができますが、絶対的なルールがあります。

- 1つの値には **常に1つの所有者** 
- 所有者がスコープを抜けると値は破棄(ドロップ)される
- 所有権は移動(ムーブ)する。

## 失敗例
### 1. 変数の移動
```rust
fn main() {
    let blackboard = "text";
    let memo = blackboard; // move

    println!("黒板: ", blackboard); // エラー
}
```

### 2. 関数へ渡す
```rust
fn greet(name: String) {
    println!("{}さんこんにちは", name);
}

fn main() {
    let name = "太郎".to_string();
    greet(name); // move
    greet(name); // エラー
}
```

## Copy/Clone
Copy/Cloneというものがあります。値を複製し、その値を新たな所有者に持たせます。

- Copyは軽量で **自動**
- Cloneは重くなる場合があり、 **手動** 

Copyが使えないケースがあります。

```rust
fn main() {
    let a = 10;
    let b = a; // 数値型は自動でコピーされる

    let x = String::from("Rust");
    let y = x; // StringはCopyが実装されていないので、ここで所有権が移る

    // println!("{}", x); // コンパイルエラー

    let z = y.clone(); // 所有権を移さない
    println!("{}", y); // OK
}
```
