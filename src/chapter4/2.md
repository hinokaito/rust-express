# 借用
やろうと思えば所有権問題はCopy/Cloneですべてを解決できます。

しかし、ただ見せたいだけの場合などではあまりにも高コストです。

`&`を付けることで、「借用」になります。これは、所有権は渡さずに参照だけさせることができます。

`&mut`を付けることで、「可変借用」になります。そのままですが、参照だけでなくその値を変更することができます。

## シンプルな借用例
```rust
fn greet(name: &String) {
    println!("{}さんこんにちは", name);
}

fn main() {
    let name = "太郎".to_string();
    greet(&name);
    greet(&name); // OK
}
```

## シンプルな可変借用例
```rust
fn change_string(s: &mut String) {
    s.push_str(", World"); 
}

fn main() {
    // 可変な変数として宣言
    let mut s = String::from("Hello");

    change_string(&mut s);

    println!("{}", s); // Hello, World
}
```
