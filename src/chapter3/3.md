# タプル
複数の型の値を1つの複合型にまとめ上げるものです。

Rustは型に厳しいため、タプルには助けられます。

割れたらtrueとその割った値が欲しいケースです。
```rust
fn can_div(left: i32, right: i32) -> (i32, bool) {
    if right == 0 {
        (0, false)
    } else {
        return (left / right, true)
    }
}

fn main() {
    let x = can_div(50, 0);
    
    if x.1 { // tuple.1 のようにしてアクセスできる。
        println!("{}", x.0);
    }
}
```
