# 型

基本的な型のみざっくり抑えましょう。スカラー型と言います。

## 符号付整数
`i8` `i16` `i32` `i64` `i128`

幅がかなり変わります。その分メモリ消費も変わります。ですが基本的にi32にCPUは最適化されているので、理由がない限りi32でOK。

デフォルト値は0

|型|最小値|最大値|
|---|---|---|
|`i8`|-128|127|
|`i32`|-2,147,483,648|2,147,483,647|
|`i128`|-170,141,183,460,469,231,731,687,303,715,884,105,728|170,141,183,460,469,231,731,687,303,715,884,105,727|

## 符号なし整数
`i8` `i16` `i32` `i64` `i128`

デフォルト値は0

|型|最小値|最大値|
|---|---|---|
|`u8`|0|255|
|`u32`|0|4,294,967,295|
|`u128`|0|340,282,366,920,938,463,463,374,607,431,768,211,455|

## 環境依存整数
`isize` `usize`

環境に依存してサイズが変わる。

デフォルト値は0

|型|説明|
|---|---|
|`isize`|符号付整数だが、幅はCPUのビット数と同じになる|
|`usize`|符号なし整数だが、幅はCPUのビット数と同じになる|

例えば配列のインデックスの型はこれになる。

## 浮動小数点数
`f32` `f64`

小数部のある数値。デフォルト値は0.0

|型|説明|範囲|精度|
|---|---|---|---|
|`f32`|単精度浮動小数点数|約 ±1.18×10⁻³⁸ ～ ±3.4×10³⁸|約7桁|
|`f64`|倍精度浮動小数点数|約 ±2.23×10⁻³⁰⁸ ～ ±1.8×10³⁰⁸|約15～16桁|

基本的に`f64`でOK

## 文字列型
`&str` `String`

今は単純に覚えましょう。

デフォルト値は`""`

**&str**
- 不変
- 参照
- メモリ上に実体はない

**String**
- 可変
- 重い
- メモリ上に実体がある

それぞれの生成や変換方法です。
```rust
fn main() {
    // &str
    let lang = "Rust";

    // String
    let lang = String::from("Python");

    // String → &str
    lang.as_str();

    // &str → String
    lang.to_string();

}
```

## 型変換
互換性に注意ですが、型変換ができます
```rust
fn main() {
    let a: i32 = 10;
    let b: f64 = 10.0;

    // このままだと計算できない
    // let c = a + b;

    // as キーワードで型変換
    let c = a as f64 + b;
    println!("{}", c);
}
```
