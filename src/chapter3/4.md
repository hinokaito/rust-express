# クレート
クレートは、簡単に言えばpythonでいうライブラリです。
違いとしては、プロジェクト単位にインストールすることです。

`cargo new`や`cargo init`して作成したRustプロジェクトの中に、Cargo.tomlというファイルがあります。そのファイルの[dependencies]の下にクレートを追加します。

または、コマンドで`cargo add`をすることで、自動で最新バージョンの物をインストールすることもできます。

## randクレートを入れてみよう
randクレートをインストールしてみましょう。

```bash
cargo add rand
```

Cargo.tomlをみると、`rand = "0.9?"`が追加されていると思います。"0.9"はバージョンを表します。

## randクレートを使ってみよう
`rand`はpythonでいう`random`です。
ランダムに生成した数値を扱ってみましょう。

`use`でインポートできます。

```rust
// * は、この場合「rand上にあるすべて」
use rand::*;

fn main() {
    let lucky_number = random_range(1..=10);

    println!("ラッキーナンバーは{}です。", lucky_number);
}
```
