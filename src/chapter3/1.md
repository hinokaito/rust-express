# 関数
関数は`fn`で宣言し、その関数名に`()`を付けて呼び出します。

```rust
fn greet() {
    println!("こんにちは");
}

fn main() {
    greet(); // こんにちは
}
```

引数を取る場合、型明示が必要で、呼び出し側はそれ以外入れられません。
次のページで型について説明するので、`i  32`とありますが、「型」という認識でいてください。
```rust
fn add(left: i32, right: i32) {
    println!("{}", left, right);
}

fn main() {
    add(10, 20); // 30
}
```

戻り値を指定できます。
戻り値も同じく型明示が必須で、それ以外の型の値は戻せません。
```rust
fn sub(left: i32, right: i32) -> i32 {
    left - right // ;がついてないので式。つまり即座に値を返す。`return left -right;` と同義。
}

fn main() {
    let x = 100;
    let y = 100;

    println!("{}", sub(x, y));
}
```
